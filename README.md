# 🏥 Smart Healthcare Appointment System  
### (Microservices-Based Web Application)

---

## 📌 Project Overview

The **Smart Healthcare Appointment System** is a microservices-based web application that enables patients to book appointments with doctors and allows doctors to manage their appointments efficiently.

The system is designed using **Node.js, Express, MongoDB, and React.js**, following a **microservices architecture** where each service operates independently and communicates through APIs.

---

## 🚀 Features

### 👤 Patient Panel
- User Registration & Login
- View Available Doctors
- Search & Filter Doctors
- Book Appointments
- View & Cancel Appointments

### 👨‍⚕️ Doctor Panel
- Doctor Registration & Login
- View Assigned Appointments
- Manage Appointment Status:
  - Accept
  - Reject
  - Complete

### ⚙️ System Features
- Microservices Architecture
- REST API Communication
- MongoDB Database Integration
- JWT Authentication
- Scalable & Modular Design
